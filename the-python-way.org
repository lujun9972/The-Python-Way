* for循环
python的 ~for~ 与C++中的for是不同的,它专指 "for each"

+ 若只想遍历list中的元素,只需要使用 =for in= 便利即可

  #+BEGIN_SRC python
    for i in ['a' , 'b' , 'c']:
        print(i)
  #+END_SRC

  #+RESULTS:
  : a
  : b
  : c

+ 若想同时遍历多个list,使用 ~for~ 与 ~zip~ 组合
  #+BEGIN_SRC python
    l1 = ["a","b","c","d","e"]
    l2 = ["A","B","C","D","E"]
    l3 = ["甲","乙","丙","丁","戊"]

    for i in zip(l1,l2,l3):
        print(i)

  #+END_SRC

  #+RESULTS:
  : ('a', 'A', '甲')
  : ('b', 'B', '乙')
  : ('c', 'C', '丙')
  : ('d', 'D', '丁')
  : ('e', 'E', '戊')

+ 若想便利list中的元素同时还需要元素的索引,使用 ~for~ 与 ~enumerate~
  #+BEGIN_SRC python
    l1 = ["a","b","c","d","e"]

    for index,value in enumerate(l1,start=1):
        print(index,value)

  #+END_SRC

  #+RESULTS:
  : 1 a
  : 2 b
  : 3 c
  : 4 d
  : 5 e

* 连续比较
这是诸多语言中少有的简便方法
#+BEGIN_SRC python
  print(3>2>1>0)
  print(1<2>0)
#+END_SRC

#+RESULTS:
: True
: True

* 交换变量
Python支持直接交换变量
#+BEGIN_SRC python
  a,b = 1,2
  print(a,b)
  a,b = b,a
  print(a,b)
#+END_SRC

#+RESULTS:
: 1 2
: 2 1
